How to install Tempalte

Before all that you have to edit your template and place your own contents by replacing old demo contents, in this case, you will need a code editor such as â€“ Notepade++, VScode, Sublime Text, etc. When template editing is done using editor, save the files and folder and go ahead to upload template files on live server.

  1. First of all, Open your Bizwheel template downloaded files, you will get a package like Template Main file.

It's possible to change the JS settings by extending the defaults from the template, that way is so much easier to update the theme changing only one JS file. all plugins activation files is here. open "Bizwheel/js/main.js" file. Here are a few samples on how to do that:

1. Changing the slider (autoplay,Slider Speed, Autoplay Speed, dots and arrows) option.

			$(".hero-slider").owlCarousel({
				loop:true,
				autoplay:true,
				smartSpeed: 700,
				margin: 0,
				nav: false,
				dots:true,
			});

2. Changing The counterup timing.

			$('.counter').counterUp({
				time: 4000,
				delay: 10,
			});

How can i use boxed layout?

Go to "Bizwheel/style.css". Then go to line number 1684 or find the "boxed layout" comments tag. Just remove Boxed Layout comments tag to use boxed layout.

How can i use template another color?

You can use 4 Amazing colors. Just remove bottom of the color link comment tag to use another color.

How can i Change google map location?

Open "Bizwheel/js/map-script.js" file. Just replace your own google map latitude and longitude address. See the red shape below to replace your google map latitude and longitude address.

How can i Replace google map API Key?

Open "Bizwheel/contact.html files" file. Find the Google Map comment tag. Just replace your own google map Api Key. See the red shape below to replace your google map Api key.
